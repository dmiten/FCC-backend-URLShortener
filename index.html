URL Shortener Microservice

dmiten | GitHub

User Stories:
  • I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.
  • If I pass an invalid URL that doesn't follow the valid http://www.example.com format, the JSON response will contain an error instead.
  • When I visit that shortened URL, it will redirect me to my original link.
    • Using API:
    You will create new record in db by adding a valid URL after https://rust-psychology.glitch.me/ with the appropriate JSON-response.
    For example:
    https://rust-psychology.glitch.me/https://www.instagram.com/netlan

    JSON-answer:
    {"original_url":"https://www.instagram.com/netlan","short_url": "https://rust-psychology.glitch.me/2760995598"}

    Contrariwise:
    https://rust-psychology.glitch.me/2760995598

    will be redirected to
    https://www.instagram.com/netlan
